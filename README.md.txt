## Project Description
A full-stack Task Management System built using the MERN stack. Users can create projects, manage tasks, assign work, track progress, and monitor performance through a dashboard.

## Features
- User Signup/Login
- Forgot Password
- JWT Authentication
- Role-based access (Admin / Member)
- Create/Edit/Delete Projects
- Create/Edit/Delete Tasks
- Mark tasks completed
- Dashboard analytics
- Task management
- Responsive UI

## Installation

### Clone repository
git clone https://github.com/Lahari0812/task-management-system-lahari.git

### Backend setup
cd backend
npm install

Create .env file:
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000

Run backend:
npm start

### Frontend setup
cd frontend
npm install
npm run dev
